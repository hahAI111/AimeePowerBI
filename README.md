# AimeePowerBI
https://www.youtube.com/watch?v=FwjaHCVNBWA

## Overview
This Power BI project analyzes job postings, applicant stages, and compensation trends through interactive dashboards to support data-driven recruitment and workforce planning.

## Data
Google Drive folder: https://drive.google.com/drive/folders/1DsLqC5OZt6fnUV8UksMhKnjWrNEG89xz?usp=drive_link

## Data Model Description
The semantic model includes these tables:

### 1) `job_postings_flat`
Detailed job posting information (title, location, source, schedule type, remote option, posting date, salary, company, and skills).

### 2) `waterfalldate`
Compensation components and amounts for analyzing pay structure and trends over time.

### 3) `Table`
Applicant pipeline stages and counts, used to evaluate recruitment funnel performance.

## Getting Started
1. Download the data from the link above.
2. Open the Power BI report file in this repository.
3. Refresh data if needed, then use filters and visuals to explore insights.
